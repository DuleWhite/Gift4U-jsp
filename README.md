# Gift4U-jsp
该项目的前端部分GITHUB项目在
https://github.com/DuleWhite/Gift4U

在前端项目的基础上添加了JSP后端

若您想要浏览添加后端后的效果可以前往
http://www.gift4u.online

若您希望运行本项目，你可能需要以下环境：
1. JDK:10.0
2. Tomcat7
3. MySQL:8.0.11
  请确保本地数据库的账户密码为：root/mysqlroot，或在代码中（src/util/DBManager.java）修改账户密码为你自己的账户密码
  并且保证本地数据库中存在名为Gift4U的数据库存在。数据库的建立文件请见项目根目录中的sql文件。
4. IDEA:2018.1及以上
